ชื่อโปรแกรม3: EvenOdd
Input:รับตัวเลขจำนวนเต็ม 1 จำนวน
Process:รับค่าตัวเลขจากผู้ใช้ แล้วตรวจสอบเศษจากการหารด้วย 2
ถ้าเศษเป็น 0 แสดงว่าเป็นเลขคู่
ถ้าไม่เป็น 0 แสดงว่าเป็นเลขคี่
Output:Even number หรือ Odd number
วิธีรันโปรแกรม:
1. Compile: javac EvenOdd.java
2. Run: java EvenOdd
3. ป้อนตัวเลข
4. โปรแกรมจะแสดงผลว่าเป็นเลขคู่หรือเลขคี่
สิ่งที่ได้เรียนรู้:
- การใช้ Scanner
- การใช้ if-else
- การใช้ตัวดำเนินการ %
